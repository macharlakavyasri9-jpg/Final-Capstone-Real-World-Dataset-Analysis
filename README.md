# Final-Capstone-Real-World-Dataset-Analysis
 COVID-19 Data Analysis – Final Capstone Project
Final Capstone – Real-World Dataset Analysis using Power BI. This project explores real-world data to create interactive dashboards and generate meaningful business insights through data visualization and analysis.
# Project Overview
This project analyzes global COVID-19 data to understand the spread, impact, and trends of the pandemic across different countries and regions. Interactive dashboards were created using Power BI to visualize cases, recoveries, and deaths.
##  Objective
- Analyze COVID-19 dataset for global insights  
- Track confirmed cases, recoveries, and deaths  
- Identify country-wise and time-based trends  
- Build interactive dashboards for data visualization  
## Tools Used
- Power BI  
- Microsoft Excel  
- Data Cleaning & Transformation Techniques  
 Python (Pandas, Matplotlib)
##  Key Dashboards
Dashboards Included
### 1️ COVID-19 Cases Trend Over Time
- Shows how confirmed cases changed over time
- Helps understand waves and spikes of the pandemic
### 2️ Global Distribution of COVID-19 Cases
- Displays worldwide spread of COVID-19 cases
- Highlights overall impact across regions
### 3 Country-wise Distribution of Cases
- Visual comparison of cases across different countries
- Helps identify heavily affected regions
### 4️ Top 5 Countries by Confirmed Cases
- Shows countries with highest confirmed cases
- Highlights major impacted countries globally  
##  Key Insights
- Identified countries with highest number of cases  
- Observed recovery trends across regions  
- Analyzed death rate variations globally  
- Found peak periods of COVID-19 spread
